# 📱 GreaterWorks PWA Installation Guide

## 🎯 Overview

GreaterWorks Church Management System is now a **Progressive Web App (PWA)** that can be installed on any device and works like a native app. This guide covers installation, features, and usage across all platforms.

## ✨ PWA Features

### 🚀 **Core PWA Capabilities**
- **Installable**: Add to home screen on mobile, desktop, and tablet
- **Offline Support**: Continue working without internet connection
- **Background Sync**: Sync data when connection is restored
- **Push Notifications**: Receive important church updates
- **App-like Experience**: Full-screen, native-like interface
- **Auto-Updates**: Automatic updates with user notification
- **Cross-Platform**: Works on iOS, Android, Windows, macOS, Linux

### 📊 **Responsive Design**
- **Mobile-First**: Optimized for touch interfaces
- **Tablet Support**: Enhanced layouts for tablet screens
- **Desktop Experience**: Full-featured desktop interface
- **Orientation Support**: Works in portrait and landscape modes
- **Safe Area Support**: Handles device notches and rounded corners

### 🔧 **Technical Features**
- **Service Worker**: Advanced caching and offline functionality
- **Web App Manifest**: Native app-like installation
- **Touch Optimized**: 44px minimum touch targets
- **Accessibility**: Screen reader support and keyboard navigation
- **Performance**: Optimized loading and smooth animations

## 📲 Installation Instructions

### 🤖 **Android Devices**

#### Chrome Browser
1. Open GreaterWorks in Chrome
2. Tap the **menu (⋮)** in the top-right corner
3. Select **"Add to Home screen"**
4. Confirm the installation
5. The app icon will appear on your home screen

#### Samsung Internet
1. Open GreaterWorks in Samsung Internet
2. Tap the **menu** button
3. Select **"Add page to"** → **"Home screen"**
4. Confirm the installation

#### Firefox Mobile
1. Open GreaterWorks in Firefox
2. Tap the **menu (⋮)**
3. Select **"Install"** or **"Add to Home screen"**
4. Confirm the installation

### 🍎 **iOS Devices (iPhone/iPad)**

#### Safari
1. Open GreaterWorks in Safari
2. Tap the **Share button** (square with arrow)
3. Scroll down and tap **"Add to Home Screen"**
4. Edit the name if desired
5. Tap **"Add"**
6. The app icon will appear on your home screen

**Note**: iOS requires Safari for PWA installation. Other browsers on iOS don't support PWA installation.

### 💻 **Desktop Computers**

#### Chrome/Chromium
1. Open GreaterWorks in Chrome
2. Look for the **install icon** (⊕) in the address bar
3. Click the install icon
4. Click **"Install"** in the dialog
5. The app will open in its own window

#### Microsoft Edge
1. Open GreaterWorks in Edge
2. Click the **menu (⋯)** → **"Apps"**
3. Select **"Install this site as an app"**
4. Confirm the installation
5. The app will appear in your Start menu

#### Firefox
1. Open GreaterWorks in Firefox
2. Click the **menu (☰)**
3. Select **"Install this site as an app"**
4. Confirm the installation

### 🖥️ **Windows 11/10**
- After installation, GreaterWorks appears in:
  - Start Menu
  - Taskbar (can be pinned)
  - Apps list
  - Desktop (optional shortcut)

### 🍎 **macOS**
- After installation, GreaterWorks appears in:
  - Applications folder
  - Dock (can be added)
  - Launchpad
  - Spotlight search

## 🛠️ Troubleshooting

### ❓ **Installation Issues**

#### "Install option not available"
- ✅ Use a supported browser (Chrome, Edge, Safari)
- ✅ Ensure you're on HTTPS
- ✅ Clear browser cache and try again
- ✅ Update your browser to the latest version

#### "App won't install on iOS"
- ✅ Use Safari browser (required for iOS PWA)
- ✅ Ensure iOS 11.3 or later
- ✅ Check available storage space
- ✅ Try restarting Safari

### 🔄 **Sync Issues**

#### "Data not syncing"
- ✅ Check internet connection
- ✅ Force close and reopen app
- ✅ Clear app cache
- ✅ Reinstall if persistent

#### "Offline mode not working"
- ✅ Ensure app was opened online first
- ✅ Check browser storage permissions
- ✅ Clear and rebuild cache
- ✅ Update to latest version

### 📱 **Performance Issues**

#### "App running slowly"
- ✅ Close other apps/tabs
- ✅ Clear browser cache
- ✅ Restart device
- ✅ Check available storage

#### "High battery usage"
- ✅ Disable background sync
- ✅ Reduce notification frequency
- ✅ Close app when not in use
- ✅ Update to latest version
